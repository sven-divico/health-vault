# extensions

## Diäten (neues Feature)
Es soll eine externe Seite geben auf der Diäten vorgeschlagen werden, die sich nach der aktuelle ehrnährung des Users Orientieren.

Z.B. wenn der Nutzer sehr viel Zucker zu sich nimmt, dann sollte die Diät Speisen mit weniger Zucker oder Getränke mit weniger Zucker vorschlagen. Eine weitere Möglichkeit wäre die Portionen zu reduzieren etc.

Der Nutzer kann den Schwierigkeitsgrad wählen, z.B. 
leicht - nur die Softdrinks rduzieren, ggf. auf einen Zielwert
Mittel - weniger Softdrinks und kleinere Portionen
Schwer - Umsetllung auf ausgewogene MAhlzeiten


## Web Menüstruktur 
Eigene Seite für jede Kategorie: Oben weiterhin der Graph mit vorbelegten Buttons
Darunter tabellarische Liste der Einträge



# Zielgewicht
der Nutzer kann sein Zielgewicht anlegen und den Zeitraum, den er sich dafür gibt


# Gewicht messen
Im Web UI soll Gewicht, Verlauf tabellarisch angezeigt werden


# Aktivität
* der Nutzer kann optional zusätzlich zu Zeit und Art verbrannte Kalorien erfassen 
* der Nutzer kann folgende Daten editieren oder auch löschen:
  * Dauer
  * Zeit & Datum
  * Aktivitätsart
  * Kalorien
